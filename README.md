# Mini CRM Automation System (Google Sheets + Apps Script)

## Overview
A fully functional CRM that automates lead management, follow-ups, dashboards, and weekly reporting. Built using Google Sheets, Google Forms, and Apps Script.

## Features
- Automated Created Date using onEdit trigger
- Daily follow-up email reminders
- Weekly PDF report automation
- Google Form → CRM auto entry
- Dashboard with charts and metrics
- One-click automation buttons
- Pivot tables and advanced analytics

## System Architecture
Google Form → CRM Sheet → Apps Script → Dashboard → Weekly Reports
